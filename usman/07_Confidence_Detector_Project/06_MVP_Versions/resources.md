# Resources — MVP Development & Testing

## Product Development
- Lean MVP approach — build minimum viable product, test with real users, iterate
- Agile sprint planning — break work into 1-2 week sprints with clear deliverables

## Testing
- Chrome DevTools Performance tab — measure FPS, identify bottlenecks
- Lighthouse — audit web app performance, accessibility, best practices
- Manual testing checklists — what to verify at each MVP version

## Deployment
- Vercel — deploy React frontend (free tier)
- Railway / Render — deploy Python backend (free tier available)
- HTTPS requirement — needed for camera access in production (Let's Encrypt for free SSL)

## PDF Generation
- html2pdf.js — convert HTML to PDF in browser
- jsPDF — generate PDFs from JavaScript
- pdfmake — PDF document generation library
