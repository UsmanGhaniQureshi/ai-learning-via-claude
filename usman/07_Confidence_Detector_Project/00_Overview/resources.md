- Mehrabian's 7-38-55 rule — body language (55%), tone (38%), words (7%) in communication
- FACS (Facial Action Coding System) — Paul Ekman's system for coding facial expressions
- Toastmasters scoring rubrics — how humans evaluate presentations professionally
