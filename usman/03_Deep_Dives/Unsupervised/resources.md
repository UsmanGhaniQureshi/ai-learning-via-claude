# Learning Resources for Unsupervised Learning

These are the best FREE YouTube channels and search terms to learn each topic. Watch in the order listed below.

---

## General Unsupervised Learning

| Channel | What to Search | Why Watch It |
|---------|---------------|--------------|
| **StatQuest** | "StatQuest unsupervised learning" | Josh Starmer's simple drawings make clustering and anomaly detection crystal clear. Start here. |
| **CodeBasics** | "CodeBasics unsupervised learning" | Full Hindi/English tutorials with Python code you can follow along with. |
| **Simplilearn** | "Simplilearn unsupervised learning" | Quick 15-20 min overviews before going deep. Good for getting the big picture. |

## Clustering Models

### K-Means Clustering
| Channel | Search Term |
|---------|------------|
| StatQuest | **"StatQuest K-Means clustering"** — THE best explanation. Watch this first! |

### DBSCAN
| Channel | Search Term |

---

## Suggested Learning Order

1. **Start with:** StatQuest "K-Means Clustering" → then StatQuest "DBSCAN"
2. **Then:** CodeBasics unsupervised learning playlist (code along!)
3. **For anomaly detection:** StatQuest "Isolation Forest" → then "Autoencoder"
4. **Practice:** Kaggle.com — search "customer segmentation dataset" and try clustering yourself

> **Tip:** For unsupervised learning, the VISUAL explanations matter a LOT. Always watch StatQuest or Normalized Nerd first to SEE how the algorithm works, then watch CodeBasics to CODE it. Understanding comes from seeing, not just reading.
