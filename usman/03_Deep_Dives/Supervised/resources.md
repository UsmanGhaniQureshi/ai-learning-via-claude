# Learning Resources for Supervised Learning

These are the best FREE YouTube channels and search terms to learn each topic. Watch in the order listed below.

---

## General Supervised Learning

| Channel | What to Search | Why Watch It |
|---------|---------------|--------------|
| **StatQuest** | "StatQuest supervised learning" | Josh Starmer explains everything with simple drawings. BEST for understanding the math without it being scary. |
| **3Blue1Brown** | "3Blue1Brown machine learning" | Beautiful animations that make abstract ideas visual. Watch when you want to SEE how things work. |
| **CodeBasics** | "CodeBasics machine learning playlist" | Full playlist in Hindi/English. Great for coding along step-by-step in Python. |

## Suggested Learning Order

1. **Start with:** StatQuest "Machine Learning Fundamentals" playlist
2. **Then:** CodeBasics "Machine Learning Tutorial Python" playlist (code along!)
3. **For visuals:** 3Blue1Brown "Neural Networks" series
4. **Practice:** Kaggle.com free courses (search "Kaggle learn machine learning")
