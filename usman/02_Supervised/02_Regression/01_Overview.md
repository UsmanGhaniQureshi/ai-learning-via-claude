# Regression

## What Is Regression?

Regression means predicting a **NUMBER.**

Not a category, not a group — an actual **number on a scale.**

> If someone asks "How much?" or "How many?" — the answer is regression.

---

## Real-World Examples

| Problem | What Goes In (Features) | What Comes Out (Number) |
|---------|------------------------|------------------------|
| **House Price** | Size, location, rooms, age | **75 lakhs** |
| **Exam Marks** | Hours studied, attendance, past grades | **82 marks** |
| **Temperature** | Month, humidity, wind speed | **35 degrees** |
| **Delivery Time** | Distance, traffic, time of day | **45 minutes** |
| **Salary** | Experience, education, skills | **80,000/month** |
| **Crop Yield** | Rainfall, fertilizer, soil type | **3.5 tons** |

---

## How to Tell If It's Regression

Ask yourself: **"Is the answer a number that could be ANYTHING on a scale?"**

| Question | Answer | Type |
|----------|--------|------|
| How much will this house cost? | 75 lakhs (a number) | **Regression** |
| Will the house sell? | Yes or No (a category) | **Classification** |
| How many marks will I get? | 82 (a number) | **Regression** |
| Will I pass or fail? | Pass or Fail (a category) | **Classification** |
| What temperature tomorrow? | 35 degrees (a number) | **Regression** |
| Will it rain tomorrow? | Yes or No (a category) | **Classification** |

**Key difference:**
- **Classification** = Answer is a WORD from a fixed list (Spam, Not Spam)
- **Regression** = Answer is a NUMBER that can be anything (75.5 lakhs, 82.3 marks, 34.7 degrees)

## Regression Models in This Folder

| Model | Best For | Difficulty |
|-------|----------|------------|
| **Linear Regression** | Simple, straight-line relationships | Easy |
| **Polynomial Regression** | Curved relationships | Medium |
| **Neural Networks** | Complex patterns with many features | Hard |

Start with **Linear Regression** — it's the simplest and most important one to understand.

## Key Terms

| Term | Meaning | Example |
|------|---------|---------|
| **Regression** | Predicting a number | Price = 75 lakhs |
| **Continuous** | The answer can be ANY number (not just whole numbers) | 75.5 lakhs, 82.3 marks |
| **Features** | The input clues | House size, rooms, location |
| **Target** | The number you want to predict | House price |
| **Error** | How far off your prediction was | Predicted 75 lakhs, actual was 78 lakhs → error = 3 lakhs |
